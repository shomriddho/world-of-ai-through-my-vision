In my Vison

AI, or artificial intelligence, is when computers are taught to do things that normally need human smarts. They can learn, solve problems, understand language, and even play games. It's like having a clever robot buddy!

Like:



  • ChatGPT

  • Github Copilot

  • Gramerly

ChatGPT is an AI-powered conversational agent. It comprehends and generates text, engaging in diverse discussions, providing information, and assisting with tasks. It's a sophisticated language model, capable of understanding context, generating human-like responses, and offering help across a wide range of topics through natural conversation.

GitHub Copilot is an AI-powered coding assistant developed by GitHub and OpenAI. It provides real-time code suggestions and completions as you write, drawing from a vast dataset of code snippets to assist developers in writing code more efficiently by generating suggestions and completing lines of code.

Grammarly is a writing assistant tool that helps users improve their writing by identifying and correcting grammar, spelling, punctuation, and style issues. It offers suggestions to enhance clarity, coherence, and overall quality of written content across various platforms and document types.
